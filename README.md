# O Jardineiro do Bairro - Landing Page

Uma landing page moderna, responsiva e interativa desenvolvida para um prestador de serviços de jardinagem e paisagismo. Este projeto foi criado para demonstrar competências em **Front-End Development**, com foco em experiência do utilizador (UX) e conversão.

## Tecnologias Utilizadas

* **HTML5:** Estrutura semântica para melhor SEO.
* **CSS3:** Design responsivo, animações customizadas e layouts com **Flexbox** e **CSS Grid**.
* **JavaScript (Vanilla):** Lógica para agendamentos, validação de formulários e feedback em tempo real ao utilizador.
* **Google Fonts:** Utilização da fonte *Open Sans* para uma leitura clara e profissional.
* **Font Awesome:** Ícones vetoriais para uma interface intuitiva.

## Funcionalidades Principais

* **Design Responsivo:** Adaptado para Desktop, Tablets e Smartphones (Mobile-first mindset).
* **Formulário de Orçamento:** Validação simples de campos e feedback visual após o envio.
* **Call to Action (CTA) Interativa:** Botão de agendamento com animação *pulse* e alteração de estado via JavaScript.
* **Navegação Suave:** Implementação de *Smooth Scroll* para uma melhor experiência de navegação entre secções.
* **Secção de Testemunhos e Serviços:** Layouts organizados em grelha para exposição clara de portefólio e prova social.

---

## Aspetos Técnicos Destacados

No desenvolvimento deste projeto, foquei-me em:
1.  **Performance:** Código limpo e sem bibliotecas externas pesadas.
2.  **UX/UI:** Uso de *overlays* informativos nos serviços e cores que remetem à natureza e profissionalismo.
3.  **Animações CSS:** Uso de `@keyframes` para guiar a atenção do utilizador para os botões de conversão.
